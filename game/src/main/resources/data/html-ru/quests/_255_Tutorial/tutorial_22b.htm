[Изменение класса Разбойника]:
Разбойник, достигший уровня 40, может во второй раз изменить класс и стать Искателем Сокровищ или Стрелком. Чтобы изменить класс во второй раз, Разбойник должен успешно пройти три квеста. Первый из этих квестов могут проходить только персонажи, достигшие уровня 35 и выше. Если вы хотите узнать больше, выберите заинтересовавший вас класс.
